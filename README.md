# CV estático — Víctor García (Data Analyst)

Esta carpeta contiene una página web estática con tu currículum en formato HTML y CSS.

Archivos:
- `index.html` — Página principal del CV. Edita los textos (Resumen, Experiencia, Educación, Proyectos, Contacto) para personalizar.
- `styles.css` — Estilos y diseño responsive.
- `cv-victor-garcia.pdf` — (opcional) Coloca aquí tu CV en PDF si quieres que el enlace "Descargar CV (PDF)" funcione.

Cómo usar

1. Abrir en el navegador (macOS):

```bash
open index.html
```

2. Personaliza `index.html`:
   - Reemplaza `victor@example.com` por tu email real.
   - Actualiza experiencias, fechas y logros.
   - Sustituye enlaces de LinkedIn y GitHub por los tuyos.
   - Si tienes un PDF, coloca `cv-victor-garcia.pdf` en la misma carpeta.

3. Imprimir/guardar como PDF: haz clic en el botón "Imprimir / Guardar como PDF" o usa la función imprimir del navegador.

Siguientes pasos sugeridos
- Añadir un archivo `cv-victor-garcia.pdf` real.
- Añadir ejemplos de notebooks y enlaces a proyectos en GitHub.
- Añadir una sección de certificaciones o cursos (ej. Coursera, DataCamp).

- Rellenar el contenido con tu experiencia real si me das fechas, empresas y logros.
